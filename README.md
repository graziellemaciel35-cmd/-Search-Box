# 🔍 Search Bar UI — Barra de Pesquisa Moderna

Este projeto apresenta uma **barra de pesquisa estilizada**, criada com HTML e CSS puro.  
O design é minimalista, elegante e inspirado em interfaces modernas, com um botão de busca circular em destaque.

Ideal para estudos de UI, microcomponentes e criação de elementos reutilizáveis em projetos web.

---

## ✨ Funcionalidades

- 🔎 Barra de pesquisa estilizada  
- 🎨 Botão circular com ícone de lupa  
- 🖤 Fundo escuro com contraste perfeito  
- 📱 Layout centralizado e responsivo  
- ⚡ Código simples e fácil de adaptar  

---

## 🛠 Tecnologias utilizadas

- **HTML5**
- **CSS3**
